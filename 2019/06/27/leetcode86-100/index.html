11. 不同的二叉搜索树(Medium)

​ 给定一个整数 n，求以 1 … n 为节点组成的二叉搜索树有多少种？

示例:

解答：

思路：

思路
标签：动态规划
假设n个节点存在二叉排序树的个数是$G(n)$，令$f(i)$为以$i$为根的二叉搜索树的个数，则
$G(n) = f(1) + f(2) + f(3) + f(4) + … + f(n)$

当$i$为根节点时，其左子树节点个数为$i-1$个，右子树节点为$n-i$，则
$f(i) = G(i-1)*G(n-i)$

综合两个公式可以得到 卡特兰数 公式
$G(n) = G(0)G(n-1)+G(1)(n-2)+…+G(n-1)*G(0)$
